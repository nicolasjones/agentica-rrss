# Propuesta de Cambio: Immersive Temporal Canvas (V5)

**ID**: `strategic-planner-v5-immersive-ux`
**Objetivo**: Transformar el Planner en una herramienta de edición visual inmersiva, priorizando el calendario como el eje central de acción y decisión.

## 📈 Valor del Cambio
1.  **Control Creativo**: El usuario puede inyectar sus propias ideas manualmente sin depender de la IA.
2.  **Contexto Temporal Profundo**: La vista semanal permite una planificación quirúrgica de horarios.
3.  **Fricción Cero**: El "Modal de Detalle" permite editar ideas y contenidos sin cambiar de página.
4.  **Mando de Eventos**: El calendario se convierte en un mapa vivo del ecosistema musical.

## 🛠️ Entregables
- [ ] **DualView Calendar**: Grid mensual + Timeline semanal.
- [ ] **Contextual Modal**: Pop-up de edición para StrategicPost (Ideas y Señales).
- [ ] **Manual Entry API**: Soporte para crear borradores de post directamente desde el frontend.
- [ ] **Scheduled Time support**: Integración de horarios en el modelo de datos.
